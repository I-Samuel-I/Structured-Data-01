# Estrutura de Dados — Atividades 1 e 2

Este repositório contém as atividades 1 e 2 desenvolvidas para a disciplina de **Estrutura de Dados**, utilizando a linguagem **Java**.

## 📚 Descrição

As atividades têm como objetivo praticar conceitos básicos de estruturas de dados, como a criação e manipulação de um **vetor personalizado**, implementando métodos para inserção, remoção, busca e outras operações.
